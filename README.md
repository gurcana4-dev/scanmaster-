# ScanMaster Pro — Çoklu QR & DataMatrix Okuyucu

## 🚀 Ücretsiz Yayınlama (GitHub Pages — 5 Dakika)

### Adım 1: GitHub Hesabı
- https://github.com adresinde hesap açın (ücretsiz)

### Adım 2: Yeni Repository Oluşturun
- Sağ üstte **"+"** → **"New repository"**
- İsim: `scanmaster` (veya istediğiniz bir isim)
- **Public** seçin
- **"Create repository"** tıklayın

### Adım 3: Dosyaları Yükleyin
- **"uploading an existing file"** linkine tıklayın
- Bu 4 dosyayı sürükleyip bırakın:
  - `index.html`
  - `manifest.json`
  - `sw.js`
  - `README.md`
- **"Commit changes"** tıklayın

### Adım 4: GitHub Pages Aktifleştirin
- Repository sayfasında **Settings** → **Pages**
- Source: **"Deploy from a branch"**
- Branch: **"main"** → **"/ (root)"**
- **Save** tıklayın

### Adım 5: Uygulamanızı Kullanın!
- 1-2 dakika bekleyin
- Adres: `https://KULLANICIADI.github.io/scanmaster/`
- Bu adresi telefonunuzda açın
- Chrome'da **"Ana ekrana ekle"** seçeneği ile uygulama olarak yükleyin

## ✅ Özellikler
- QR Code + DataMatrix + Code128 + EAN okuma
- Aynı karede çoklu kod algılama
- Koli takibi (X adette bir koli tamamlandı uyarısı)
- Referans kod listesi yükleme (Excel/CSV/TXT)
- Doğru / Yanlış / Tekrar tespit
- Ses + titreşim geri bildirimi
- Excel'e dışa aktarma (4 sayfa: Tüm okumalar, Doğru kodlar, Hatalar, Özet)
- Eksik kodları raporlama
- Manuel kod girişi
- Fener (flashlight) desteği
- PWA — uygulama olarak yüklenebilir
- Çevrimdışı çalışabilir
